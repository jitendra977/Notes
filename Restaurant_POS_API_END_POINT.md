# API Endpoints

## Customer API
- **[GET]** /api/customers
  - Retrieve all customers
- **[POST]** /api/customers
  - Create new customer
- **[GET]** /api/customers/{id}
  - Retrieve specific customer
- **[PUT]** /api/customers/{id}
  - Update specific customer
- **[DELETE]** /api/customers/{id}
  - Delete specific customer
